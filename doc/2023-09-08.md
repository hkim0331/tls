## lat?

引数 xs が lat
(アトムからなるリスト。リストを要素に持たないリスト)
かどうかを判定する。

??? に入るのはなんでしょう？

```scheme
(define lat?
  (lambda (xs)
    (cond
      ((null? xs) #t)
      ((atom? (first xs) (??? (rest xs)))
      (else #f)))))
```

## even?

n は偶数ですか？

??? に入るのはなんでしょう？

```scheme
(define even?
  (lambda (n)
    (cond
    ((eq? n 0) #t)
    ((eq? n 1) #f)
    (else (??? (- n 2))))))
```
